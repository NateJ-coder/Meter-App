# Legacy Utility Dash Assets

This folder contains extracted legacy Utility Dash workbook material that was pulled out of the decrypted Excel file for reference and migration work.

## Structure

- `vba-source/`: exported VBA modules and workbook object classes
- `workbook-exports/`: decrypted workbook files kept as source artifacts
- `notes/`: working notes about extraction, mapping, and migration observations

These files are reference assets for analysis and migration. They are not part of the live web app runtime.
