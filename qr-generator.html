QR Codes

One QR code per scheme — attach it to the building entrance. Scanning shows live scheme info and opens the reading walkthrough when a cycle is active.

Filter

Scheme QR Codes

How to use

  1. One QR code is generated per scheme. Print it and attach it to the entrance of the building or scheme notice board.
  2. When a reader scans the code with their phone, they see the scheme overview — unit count, last readings, and any open flags.
  3. If a reading cycle is currently open, the Begin Reading button activates and walks the reader through every unit in order.
  4. If no cycle is open, the button is greyed out and the reader can still view current meter info.
Note: No login is required to scan and view scheme information. The reading capture form only appears when an active cycle is open.
